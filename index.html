About Job Hunter

Job Hunter is a powerful toolkit designed to help job seekers find opportunities faster and more efficiently. The web app allows you to search multiple platforms at once, while the browser extension provides additional features:

Web App Features

  • Search multiple job platforms with a single click
  • Discover hidden job opportunities in documents and online content
  • Use AI to generate tailored job search queries

Browser Extension Features

  • Save and track job listings from any website
  • Automatically extract job details when browsing listings
  • Generate tailored CVs and cover letters for specific jobs
  • Manage your job application progress in one place

Get the Job Hunter Extension

The Job Hunter browser extension enhances your job search with powerful features:

  • Save jobs from any website with a single right click
  • Track applications and keep everything organized
  • Auto-extract job details from listings you browse
  • Generate personalized applications with AI assistance

Installation Steps:

  1. Download the extension package below
  2. Extract the ZIP file to a folder on your computer
  3. Open your browser's extension management page
  4. Enable "Developer mode"
  5. Click "Load unpacked" and select the extracted folder
  6. The Job Hunter icon should appear in your browser toolbar
